# GLO3004_H18_TP2

# Configuration

Pour configurer le paramètre de CValve et CEtiquetage, modifier la variable static MAX_CONCURRENT dans la classe Confiturerie.
Pour configurer le nombre de bocal de chaque type, modifier la variable static N dans la classe Confiturerie.
Pour configurer une rupture de stock sur 1 bocal, mettre le dernier paramètre du constructeur a true.

# Utilisation

Il suffit de lancer Confiturerie.java . Les actions sont afficher dans la console dans l'ordre ou ils surviennent.
